My recommendation is **B, not C**. **A** is the cleanest estimand, but it multiplies rewrite, QC, and annotation cost right after you already fixed the larger edit-magnitude confound. Entity-triggered recall is now a mechanism probe, so run a **Tier1 masking ablation on the 300 anchors** as a separate block, and pre-register escalation to **A** only if the signal is clearly non-trivial. **C** is the wrong move because making masking the main control would change the task itself and break comparability with the core E3/E4 results.

For masking, use **natural substitution as the primary treatment**. Replace names with fluent, type-consistent descriptors such as `某监管机构`, `某白酒龙头`, `某半导体公司`. The rule-based `[实体N]` variant is too behavior-changing: it injects rare placeholders, degrades fluency, and interacts badly with evidence-copying prompts. I would keep `[实体N]` only as a **small robustness stress test** or upper-bound “memory-cue knockout,” not as the headline estimate. The existing `key_entities` field is a good seed list for targeted naturalization.

Yes, masking **does interact with the current target-conditioned prompt design**. If `Target: 贵州茅台` remains visible, the target field itself can still serve as the retrieval key; if you mask the target, `judge FOR {target}` and `target_echo` break. So the masking ablation needs a **separate prompt family**: rewrite the article so the focal referent is marked as `[TARGET]` or a natural role phrase like `该公司`, and ask the model to judge the **marked target**, not to echo the raw name.

For decomposed tasks, masking is **not equally sensible**. It makes sense for **direct prediction**, **sentiment**, and probably **novelty**, because the relevant semantic cues survive anonymization. It is **not a blanket fit for authority**: masking provenance-bearing names destroys the object you are measuring. For authority, either skip masking or use **restricted masking** that preserves official/source mentions and masks only focal-company / peer-company identities.

A smarter probe than full masking is a **cue-location ladder**: compare `(i)` raw names, `(ii)` target-pointer only, `(iii)` non-target entity naturalization, and `(iv)` an entity-swap placebo using a same-type/same-sector substitute. That identifies whether recall is keyed by the target field, article-body entities, or broader event semantics.
